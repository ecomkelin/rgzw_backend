## 模块覆盖

### Label 模块测试覆盖
- **服务层 (service.test.js)**: 
  - 业务逻辑验证
  - 数据库操作验证
  - 权限控制验证
  - 软删除逻辑验证

- **控制器 (controller.test.js)**:
  - API 端点响应验证
  - 错误处理验证
  - HTTP 状态码验证

- **工具函数 (softDeleteUtils.test.js)**:
  - 软删除/恢复逻辑
  - 永久删除逻辑
  - 已删除列表查询

- **API 集成测试 (api.test.js)**:
  - 完整 API 工作流
  - 真实数据库操作
  - 认证授权验证

- **端到端测试 (e2e.test.js)**:
  - 完整用户操作流程
  - 标签创建到删除的生命周期
  - 软删除和恢复功能
